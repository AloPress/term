# Nutiseadmete oskussõnastik

Nutiseadmete oskussõnastik on projekt mille eesmärk on eestistada nutiseadmete valdkonnaga seotud terminoloogiat.

Projekt katab esialgu järgmised valdkonnad
- Seadme haldus
- Kriminalistika
- Küberturvalisus
- Tarkvara arendus
- Telekommunikatsioon
- Operatsioonisüsteem
